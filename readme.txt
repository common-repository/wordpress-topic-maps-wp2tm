Semantic Web plugin which adds XML Topic Maps (XTM) feed support to WordPress blogs.

== Description ==

The WP2TM plugin adds XML Topic Maps (XTM) "feed" support to WordPress blogs. 

**[Download now!](http://www.topicobserver.com/wp2tm/download/)**

WP2TM may enable the sharing and merging of knowledge across WordPress blogs.
XTM exports produced by WP2TM can be merged using standard compliant topic maps software.

Requires PHP5.

[Support](http://www.topicobserver.com/wp2tm/support/) |
[Version History](http://www.topicobserver.com/wp2tm/category/releases/) |
[FAQ](http://www.topicobserver.com/wp2tm/faq/)

Some features:

* XML Topic Maps (XTM) export of individual posts and/or entire blog
* Maintains post <-> author, post <-> tag and post <-> category relationships
* Uses Published Subject Indicators (PSI) for common concepts
* Control via Wordpress' admin panel
* Relate blog tags to other tags via admin panel, for use in XTM export

== Installation ==

1. Unzip into your `/wp-content/plugins/` directory. If you're uploading it, make sure to upload
the top-level folder. Don't just upload all the files and put them in `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your WP2TM options (*Options - WP Topic Maps*)
4. That's it!
